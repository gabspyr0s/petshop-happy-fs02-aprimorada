# PetShop Happy
site para um petshop (Fase 2).
Esta versão inclui Bootstrap, CSS e JavaScript para tornar o site mais dinâmico.

Arquivos principais:
- index.html
- produtos.html
- servicos.html
- contato.html
- cadastro.html
- styles.css
- scripts.js

Instruções:
- Coloque todos os arquivos em uma mesma pasta.
- Abra `index.html` no navegador.
